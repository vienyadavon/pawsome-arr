## Development Notes:
This project was built with AI assistance (Claude by Anthropic) for 
code review, architecture guidance, and pair programming support. 
All design decisions, implementation, debugging, and domain framing 
are my own work.
Portions of the csv output scripts were generated using Google Gemini, 
an AI Coding Assistant. The AI was used to assist with randomized spreadsheet 
generation of simulated sales data in order to closely mimic those used by 
RevOps or Sales Ops departments.
The resulting code was subsequently reviewed, tested, and integrated by me.
